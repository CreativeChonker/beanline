# Point of Sale (POS) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a staff-facing register (`/pos`) so an owner or staff member can ring up a walk-in, in-person sale — pick items from a card grid, see a running total, choose Cash or Card, complete the sale — reusing the interaction pattern already built for customer self-ordering.

**Architecture:** Extend the existing `orders` table (from the tenancy sub-project) to support two mutually-exclusive order shapes: a customer self-order (`user_id` set) or a staff-rung-up walk-in sale (`staff_user_id` set), enforced by a database CHECK constraint. `models/orders.js` grows to handle both. `/pos` reuses `/:shopSlug/order`'s card-grid + running-total client-side pattern, adapted for the staff app-shell and a payment-method choice instead of a shop slug.

**Tech Stack:** Same as prior sub-projects — Node.js, Express, EJS, `pg` (raw SQL, no ORM), `node:test` + `supertest` against real Postgres.

## Global Constraints

- Every order is either a customer self-order (`user_id` set, `staff_user_id` NULL, `payment_method` NULL) or a POS walk-in sale (`user_id` NULL, `staff_user_id` set, `payment_method` set) — never both set, never neither. Enforced by a database CHECK constraint (`orders_customer_xor_staff`), not just application code.
- `/pos` is gated by `requireAuth` + `requireRole('owner', 'staff')` — same access as `/dashboard`, no shop slug in the URL, scoped via `session.user.shopId`.
- A POS sale always starts `status = 'completed'` (it's already paid and handed over), unlike a customer self-order which starts `'received'`.
- `payment_method` is `'cash'` or `'card'`, required on every POS sale, validated server-side — never defaulted silently.
- An unavailable menu item's id submitted to `POST /pos` is silently excluded from the sale, identical to the existing customer order route's behavior — a POS sale can't ring up something the shop can't fulfill either.
- No real payment processing — `payment_method` is a bookkeeping field only, no card reader/Stripe/charge of any kind.
- No ORM — raw SQL via the `pg` driver.
- Every new piece of logic gets a test before being wired into `server.js`.

## Task 1: Extend the `orders` schema for walk-in sales

**Files:**
- Modify: `db/schema.sql` (append ALTER statements after the existing `orders` table block)

**Interfaces:**
- Produces: `orders.user_id` becomes nullable; new nullable `orders.staff_user_id` (FK to `users`) and `orders.payment_method` (TEXT) columns; a `orders_customer_xor_staff` CHECK constraint. Consumed by Task 2 (model).

**Why ALTER, not just CREATE TABLE IF NOT EXISTS:** `orders` already exists in every environment this app runs in (dev, test, and eventually production) from the tenancy sub-project. `migrate.js` re-runs the entire `schema.sql` file on every invocation, so these statements must be idempotent (safe to run repeatedly) the same way the existing `CREATE TABLE IF NOT EXISTS` statements are.

- [ ] **Step 1: Append the migration to `db/schema.sql`**

Add after the existing `CREATE TABLE IF NOT EXISTS orders (...)` block and its indexes (keep the `menu_items` table and its index, which come after `orders` today, at the end — add this new block between `orders`'s indexes and the `menu_items` table):

```sql
ALTER TABLE orders ALTER COLUMN user_id DROP NOT NULL;
ALTER TABLE orders ADD COLUMN IF NOT EXISTS staff_user_id INTEGER REFERENCES users(id);
ALTER TABLE orders ADD COLUMN IF NOT EXISTS payment_method TEXT;

DO $$
BEGIN
  IF NOT EXISTS (
    SELECT 1 FROM pg_constraint WHERE conname = 'orders_customer_xor_staff'
  ) THEN
    ALTER TABLE orders ADD CONSTRAINT orders_customer_xor_staff CHECK (
      (user_id IS NOT NULL AND staff_user_id IS NULL) OR
      (user_id IS NULL AND staff_user_id IS NOT NULL)
    );
  END IF;
END $$;
```

```js
async function createOrder(queryable, { userId = null, staffUserId = null, shopId, items, total, status = 'received', paymentMethod = null }) {
  const result = await queryable.query(
    `INSERT INTO orders (user_id, staff_user_id, shop_id, items_json, total, status, payment_method)
     VALUES ($1, $2, $3, $4, $5, $6, $7)
     RETURNING id, created_at`,
    [userId, staffUserId, shopId, JSON.stringify(items), total, status, paymentMethod]
  );
  return result.rows[0];
}

async function getOrdersForShop(queryable, shopId) {
  const result = await queryable.query(
    `SELECT orders.id, orders.items_json, orders.total::float8 AS total, orders.status, orders.created_at,
            orders.payment_method,
            customer.name AS customer_name, customer.email AS customer_email,
            staff.name AS staff_name
     FROM orders
     LEFT JOIN users customer ON customer.id = orders.user_id
     LEFT JOIN users staff ON staff.id = orders.staff_user_id
     WHERE orders.shop_id = $1
     ORDER BY orders.created_at DESC`,
    [shopId]
  );
  return result.rows.map((o) => ({ ...o, items: JSON.parse(o.items_json) }));
}

module.exports = { createOrder, getOrdersForShop };
```

          card.style.display = (activeCat === 'all' || card.dataset.category === activeCat) ? '' : 'none';
        });
      });
    });

    document.querySelectorAll('.menu-card').forEach(card => {
      card.addEventListener('click', () => {
        const id = card.dataset.id;
        const name = card.dataset.itemName;
        const price = parseFloat(card.dataset.price);
        if (cart[id]) {
          cart[id].qty += 1;
        } else {
          cart[id] = { name, price, qty: 1 };
        }
        renderBill();
        card.classList.add('just-added');
        setTimeout(() => card.classList.remove('just-added'), 400);
      });
    });

    document.querySelectorAll('.payment-choice').forEach(choice => {
      choice.addEventListener('click', () => {
        document.querySelectorAll('.payment-choice').forEach(c => c.classList.remove('selected'));
        choice.classList.add('selected');
        selectedPayment = choice.dataset.method;
        document.getElementById('payment-method-input').value = selectedPayment;
        updateProcessButton();
      });
    });

    function removeFromCart(id) {
      delete cart[id];
      renderBill();
    }

    function updateProcessButton() {
      const hasItems = Object.keys(cart).length > 0;
      document.getElementById('process-btn').disabled = !(hasItems && selectedPayment);
    }

    function renderBill() {
      const billList = document.getElementById('bill-list');
      const hiddenInputs = document.getElementById('hidden-inputs');
      const ids = Object.keys(cart);

      hiddenInputs.innerHTML = '';
      if (ids.length === 0) {
        billList.innerHTML = '<p class="bill-empty">Tap an item to add it.</p>';
        document.getElementById('bill-count').textContent = '0';
        document.getElementById('bill-total').textContent = '$0.00';
        updateProcessButton();
        return;
      }

      let total = 0;
      let count = 0;
      billList.innerHTML = ids.map(id => {
        const item = cart[id];
        total += item.price * item.qty;
        count += item.qty;
        return `<div class="receipt-line bill-item">
          <span class="qty-name">${item.name} &times;${item.qty}</span>
          <span class="dots"></span>
          <span>$${(item.price * item.qty).toFixed(2)}</span>
          <button type="button" class="remove" onclick="removeFromCart('${id}')">remove</button>
        </div>`;
      }).join('');

      ids.forEach(id => {
        const input = document.createElement('input');
        input.type = 'hidden';
        input.name = 'qty_' + id;
        input.value = cart[id].qty;
        hiddenInputs.appendChild(input);
      });

      document.getElementById('bill-count').textContent = count;
      document.getElementById('bill-total').textContent = '$' + total.toFixed(2);
      updateProcessButton();
    }
  </script>
</body>

app.post('/pos', requireAuth, requireRole('owner', 'staff'), async (req, res, next) => {
  const { paymentMethod } = req.body;
  try {
    const availableItems = await menuItems.getMenuItemsForShop(db, req.session.user.shopId, { availableOnly: true });
    const items = [];
    let total = 0;
    for (const item of availableItems) {
      const qty = parseInt(req.body['qty_' + item.id], 10) || 0;
      if (qty > 0) {
        items.push({ name: item.name, qty, price: item.price });
        total += qty * item.price;
      }
    }

    if (items.length === 0) {
      return res.render('pos', { menu: availableItems, error: 'Please select at least one item.' });
    }
    if (!['cash', 'card'].includes(paymentMethod)) {
      return res.render('pos', { menu: availableItems, error: 'Please choose a payment method.' });
    }

    const created = await orders.createOrder(db, {
      staffUserId: req.session.user.id,
      shopId: req.session.user.shopId,
      items,
      total,
      status: 'completed',
      paymentMethod,
    });

    res.render('pos-receipt', {
      sale: {
        order_id: created.id,
        staff_name: req.session.user.name,
        lineItems: items,
        total: total.toFixed(2),
        payment_method: paymentMethod,
        created_at: created.created_at,
      },
    });
  } catch (err) {
    next(err);
  }
});
